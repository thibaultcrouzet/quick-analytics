A simple plugin that helps you to integrate quickly your analytics tracking IDs.

== Description ==

This plugin allows you to add the analytics tracking for : Google Analytics, Yandex Metrika, Mixpanel, Kissmetrics, Woopra, Heap, GoSquared, Statcounter.
More will be added later.

== Installation ==

1. Download the plugin and unzip it.
2. Upload the folder quick-analytics/ to your /wp-content/plugins/ folder.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the tracking IDs you need.
5. Installation finished.
